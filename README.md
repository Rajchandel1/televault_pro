# Televault Pro

Decentralized cloud storage via Telegram Bot.

## Quick start

- Install dependencies: `npm install`
- Create a `.env` file from your environment template (contains Supabase + Telegram credentials).
- Run locally: `npm run dev` (requires `nodemon`) or `npm start`.

## Project layout

- `server.js` — app entry
- `public/` — frontend assets
- `src/` — server source (routes, services, bot handlers, utils)
